# 01-任务确认

## 目标
开发“AI春联工坊”并可部署到个人网站：输入主题后生成上联/下联/横批与解释，并支持生成可下载海报图。

## 关键约束
- 必须由 Codex CLI 完成开发实现（主代理不替代写代码）。
- 文案生成与图片生成都走 AI Builder MCP/接口能力。
- 禁止使用用户个人 API Key（尤其 OpenAI Key）。
- token 仅通过环境变量读取，禁止写死到代码与日志。

## 分阶段
- M1: 可运行骨架 + 文案生成 + 页面展示复制
- M2: 海报图生成 + 下载
- M3: 移动端优化 + 自测与交付

## 验收（初版）
1. 页面可打开并提交主题。
2. 可返回结构化春联内容（上/下/横批/解释）。
3. 文案调用路径不使用用户私钥。
4. 有基础错误处理和失败提示。
5. 提供启动与自测命令。
