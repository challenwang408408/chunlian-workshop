# 移动端检查清单（M3）

适用视口：`390x844`（可同步复测 `375x812`）。

## 页面布局
- [ ] 输入区无横向滚动，字段完整可见。
- [ ] 结果区文本不溢出，长文本可自动换行。
- [ ] 海报预览区域宽度自适应容器，无横向溢出。

## 交互可用性
- [ ] 主要按钮高度 `>=44px`：生成春联、生成海报、复制、下载。
- [ ] 生成春联时：提交按钮禁用，显示加载文案。
- [ ] 生成海报时：海报按钮禁用，显示加载文案。
- [ ] 不可用操作（无结果时复制、无海报时下载）处于禁用态。

## 错误与重试
- [ ] 参数错误提示可读，用户可按提示修改后重试。
- [ ] 网络/上游失败提示可读，建议稍后重试。
- [ ] 错误提示包含请求编号（requestId），便于排障。

## 回归建议
- [ ] 在慢网环境下验证加载态与禁用态是否生效。
- [ ] 在海报生成失败后，确认春联文本仍可复制与继续使用。
